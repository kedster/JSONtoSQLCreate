Example JSON Structures

Flat JSON

{
  "id": 1,
  "name": "John Doe",
  "email": "john@example.com"
}

Nested JSON

{
  "user": {
    "id": 1,
    "name": "Jane Smith"
  },
  "posts": [
    {
      "title": "First Post",
      "content": "Hello World"
    },
    {
      "title": "Second Post",
      "content": "More text"
    }
  ]
}
